## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

```bash
git clone https://github.com/AryanSingh4510/ePorfume.git
cd ePorfume
npm install
```

### Running the App

```bash
npm run dev
```

The app will be available at `http://localhost:5173/` (default Vite port).
